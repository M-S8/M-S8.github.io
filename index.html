Python • SQL • Statistical Analysis

Customer Churn Prediction – Understanding When Customers Leave

Python • Survival Analysis

Instead of predicting churn as a binary event, I modelled it as a timeline. Survival analysis helped reveal when churn risk spikes and which customer groups require intervention.

  • Built survival curves and hazard functions for customer cohorts.
  • Identified high-risk periods for targeted retention.
  • Created interpretable outputs for non-technical stakeholders.

Bayesian Technician Ranking – Fair Performance Evaluation

Python • Bayesian Modelling

Technician performance varies by workload, geography, and noise. I built a Bayesian hierarchical model that estimates “true” performance while accounting for uncertainty.

  • Designed a hierarchical Bayesian model capturing technician & region variability.
  • Generated posterior distributions to rank fairly with uncertainty bands.
  • Provided leadership with a transparent and defensible ranking method.
