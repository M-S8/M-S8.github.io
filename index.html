Revenue Anomaly Detection – Finding Hidden Signals

Python · SQL · Statistical Analysis

Built an automated weekly anomaly detection system across 50+ regions using IQR-based modelling to flag unexpected revenue dips early for business review.

  • Automated detection of unusual revenue behaviour instead of manual spreadsheet checks.
  • Delivered clear summary views to regional stakeholders for faster action.

Customer Churn Prediction – Understanding When Customers Leave

Python · Survival Analysis

Modelled churn as a time-to-event problem using survival analysis to identify when churn risk peaks and which customer cohorts need proactive outreach.

  • Built survival curves and hazard functions for key customer segments.
  • Highlighted high-risk periods to inform targeted retention strategies.

Bayesian Technician Ranking – Fair Performance Evaluation

Python · Bayesian Modelling

Used Bayesian hierarchical modelling to estimate “true” technician performance while capturing uncertainty and variation across regions.

  • Modelled technician- and region-level effects in a single probabilistic framework.
  • Produced posterior distributions instead of single scores for more honest rankings.

Python · Foursquare API · Clustering

Analysed venues in Nice, France to cluster neighbourhoods and identify high-opportunity areas for a new business concept.
