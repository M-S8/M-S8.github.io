Revenue Anomaly Detection – Finding Hidden Signals

Python · SQL · Statistical Analysis

Revenue drops rarely happen “out of nowhere”. I built an automated monitoring system that scans weekly revenue across 50+ regions and flags unusual dips early enough for teams to react.

  • Implemented IQR-based anomaly detection on regional revenue time series.
  • Replaced manual spreadsheet checks with a reproducible Python + SQL pipeline.
  • Produced clear summary tables and narratives that guided regional follow-up actions.
CH

Customer Churn Prediction – Understanding When Customers Leave

Python · Survival Analysis

Instead of treating churn as a simple yes/no prediction, I modelled it as a timeline. Survival analysis helped identify when churn risk peaks and which customer groups need proactive attention.

  • Built survival curves and hazard functions for multiple customer cohorts.
  • Surfaced high-risk periods to inform targeted retention strategies.
  • Focused on interpretability so non-technical stakeholders could see not just who churns, but when and why.
BT

Bayesian Technician Ranking – Fair Performance Evaluation

Python · Bayesian Modelling

Ranking technicians by performance is tricky when workloads and opportunities differ. I used Bayesian hierarchical modelling to estimate “true performance” while showing the uncertainty behind each estimate.

  • Modelled technician and region-level variation using a hierarchical Bayesian framework.
  • Generated posterior distributions instead of single-point scores to capture uncertainty.
  • Provided leadership with a transparent and defendable ranking method for training and incentives.

Python · Foursquare API · Clustering

Cities tell stories through where businesses cluster. I analysed neighbourhoods in Nice, France, to identify areas with attractive venue patterns for a new business concept.

  • Collected venue data using the Foursquare API for multiple neighbourhoods.

  • Used geospatial visualisation to highlight zones with strong opportunity signals.
