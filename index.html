Revenue Anomaly Detection System

Python • SQL • Statistical Analysis

Built an automated anomaly-detection framework to monitor weekly revenue across regions and identify unusual patterns without manual checks.

  • Implemented IQR-based outlier detection on regional revenue time series to surface underperforming areas.
  • Automated data extraction and transformation using SQL, reducing manual monitoring effort.
  • Provided stakeholders with a repeatable process to quickly investigate and act on revenue issues.

Customer Churn Prediction with Survival Analysis

Python • Survival Analysis • SQL

Modelled time-to-cancellation for home warranty customers to support proactive retention strategies.

  • Applied survival-analysis techniques to estimate churn curves and identify higher-risk customer segments.
  • Combined transactional, tenure, and behavioural features into a structured modelling dataset.
  • Generated insights that informed targeted retention offers and prioritisation of outreach efforts.

Bayesian Technician Ranking Model

Python • Bayesian Modelling • Hierarchical Statistics

Developed a Bayesian hierarchical model to produce fair, uncertainty-aware rankings of technician performance.

  • Modelled technician-level profitability and success metrics while accounting for data sparsity and variance.
  • Generated posterior distributions for each technician, enabling comparison with credible intervals.
  • Delivered a ranking framework to support decisions on training, incentives, and workforce planning.

ETL Pipeline for Parts Pricing

Python • Web Scraping • Snowflake

Designed an ETL workflow to collect and centralise appliance part prices from online sources for competitive analysis.

  • Scraped pricing data from publicly available websites, handling pagination and HTML variability.
  • Normalised and cleaned raw data in Python before loading into Snowflake.
  • Enabled downstream analysis of price dispersion and market positioning.

Geospatial Business Feasibility Study – Nice, France

Python • Foursquare API • Geospatial Analysis

Evaluated potential locations for a new business in Nice by analysing venue density, competition, and neighbourhood profiles.

  • Fetched and processed venue data using the Foursquare API for multiple neighbourhoods.
  • Used clustering to segment areas by venue composition and identify high-opportunity zones.
  • Visualised results on geospatial maps to support location selection decisions.

A/B Testing Analysis Framework

Python • Statistics • Experimentation

Created a reusable framework to evaluate A/B test performance and quantify the impact of new features or offers.

  • Automated calculation of lift, confidence intervals, and p-values for experiment groups.
  • Incorporated bootstrapping to better understand result stability and uncertainty.
  • Packaged the workflow into a repeatable script that can be applied to multiple experiments.
